# User Profile

The user is a portfolio manager or investment professional who manages one or more investment portfolios.

## Communication Preferences

- Prefers structured, concise updates
- Values materiality assessment (what matters vs what is noise)
- Wants to understand the "why" behind events, not just the "what"
- Expects professional-grade analysis
- May ask follow-up questions to drill deeper

## Access Level

- Full read access to all portfolio data
- Can upload portfolio updates
- Can request ad-hoc analysis
- Can acknowledge alerts
- Can request digests on demand
